# Cataleya v4

E-commerce de indumentaria femenina. Next.js 15 + Supabase + Tailwind.

## Variables en Vercel
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- NEXT_PUBLIC_WHATSAPP

## Rutas
- `/` Home
- `/categoria/[slug]` Listado por categoría
- `/producto/[id]` Detalle producto
- `/carrito` Carrito
- `/[slug]` Páginas estáticas (envíos, cambios, etc.)
- `/admin/login` Login admin
- `/admin/*` Panel admin
